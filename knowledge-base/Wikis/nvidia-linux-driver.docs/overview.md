# NVIDIA Linux Driver README Overview

Source: NVIDIA Accelerated Linux Graphics Driver README and Installation Guide, Linux x86_64 driver 580.105.08.

This directory keeps selected official README chapters that are useful for Linux desktop, Wayland/X11, PRIME, power management, driver installation, and troubleshooting questions.

## Audio Support

Many NVIDIA GPUs support embedding an audio stream in HDMI and DisplayPort signals. In most cases, the GPU contains a standard HD-Audio controller, for which there is a standard ALSA driver. NVIDIA's GPU HDMI audio document has more details: <https://download.nvidia.com/XFree86/gpu-hdmi-audio-document/>.
