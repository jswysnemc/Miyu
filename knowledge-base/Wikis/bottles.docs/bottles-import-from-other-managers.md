# Import from other managers

## Import from other managers

Currently we support migration from these applications:

* Bottles V1
* Lutris
* PlayOnLinux

### How to import

You can access the import tools from the Bottles main menu by selecting the "Import/export" option.

![Bottles - Main Menu](/files/LCPyIedVkE758IeA8r39)

The "Importer" window will open.

![Bottles - Importer](/files/X5A2DJnRO6zPBmjdsWuP)

Clicking on the Refresh button (![Bottles - Importer - Refresh Button](/files/wjuEn3fnofuVtcfxMAn8)) will launch a search for wineprefixes available for import.

Button ![Bottles - Importer - Import Button](/files/CJaP4xDxImiFWvAs9y92) allows you to import bottle backups created earlier ([Backups & Duplicate](/bottles/backups.md)).
