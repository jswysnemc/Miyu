### [Overview]

[![Msm.jpg](/images/thumb/5/59/Msm.jpg/375px-Msm.jpg)](//wiki.manjaro.org/index.php?title=File:Msm.jpg)

[](//wiki.manjaro.org/index.php?title=File:Msm.jpg "Enlarge")

The Manjaro Settings Manager offers you a series of settings, which are either enabled by Manjaro (i.e. installation of multiple kernels) or are missing from at least some of popular [Desktop Environments and Window Managers](//wiki.manjaro.org/index.php?title=Desktop_Environments "Desktop Environments").

Manjaro Settings Manager is under development and might offer even more settings than illustrated on the left.

### [Language]

[![Locale setting.png](/images/thumb/a/a5/Locale_setting.png/375px-Locale_setting.png)](//wiki.manjaro.org/index.php?title=File:Locale_setting.png)

[](//wiki.manjaro.org/index.php?title=File:Locale_setting.png "Enlarge")

The system language can be added, changed, and removed in this window.

If you want to have different languages for System, individual user and the Linux console, read how on [Locale](//wiki.manjaro.org/index.php?title=Locale "Locale")

### [Kernel]

[![Kernel screen.png](/images/thumb/6/69/Kernel_screen.png/375px-Kernel_screen.png)](//wiki.manjaro.org/index.php?title=File:Kernel_screen.png)

[](//wiki.manjaro.org/index.php?title=File:Kernel_screen.png "Enlarge")

This window enables the user to easily manage and change all available kernels for Manjaro. Running and installed kernels can be easily identified. Read more on the [Manjaro Kernels](//wiki.manjaro.org/index.php?title=Manjaro_Kernels "Manjaro Kernels") page.

### [Time and Date]

[![Time date screen.png](/images/thumb/5/58/Time_date_screen.png/375px-Time_date_screen.png)](//wiki.manjaro.org/index.php?title=File:Time_date_screen.png)

[](//wiki.manjaro.org/index.php?title=File:Time_date_screen.png "Enlarge")

Some Desktop Environments do not offer an easy way to edit time and date (i.e. XFCE).

This window enables the user to do that.

### [Hardware Detection]

[![Hardware screen.png](/images/thumb/c/c7/Hardware_screen.png/375px-Hardware_screen.png)](//wiki.manjaro.org/index.php?title=File:Hardware_screen.png)

[](//wiki.manjaro.org/index.php?title=File:Hardware_screen.png "Enlarge")

This window offers an easy way install free/non-proprietary/open-source and non-free/proprietary/closed-source graphics card drivers by the click of a button.

If Manjaro did not install a wireless network card driver during installation, it can be installed here as well.
