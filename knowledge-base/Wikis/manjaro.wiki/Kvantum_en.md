# [Overview]

Kvantum is customizable SVG-based theme engine for Qt5 that comes with a variety of built-in styles, including versions of some of popular GTK+ themes such as Adapta, Arc, Ambiance, Materia.

# [Installation]

To install Kvantum install the package `kvantum-manjaro` in your favorite package manager or using the command:

    pamac install kvantum-manjaro

\

**Note**

------------------------------------------------------------------------

In order to use kvantum you may have to enable it for your desktop or window manager - either manually or through your DE\'s settings.

\

# [Configuration]

**Kvantum Manager** can be used to manage your kvantum themes and settings.\
This also includes \'hacks\' such as transparent Dolphin view, blur, icon sizes, and more.
