## [][Huh, what is "mounting"???]

"Mounting" is making a drive available to the system.

-   I.E. If you insert a USB stick into a Windows machine, it will get the F: drive letter automatically Windows will automatically mount the USB drive as F:
-   You need to know that is happening automatically under Windows, because under Linux it's better to do this manually as Linux provides more flexibility in how and where you mount partitions! :+1:
