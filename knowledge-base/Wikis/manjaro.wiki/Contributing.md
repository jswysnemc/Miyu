## [How to contribute Manjaro Wiki]

### [Registering]

[![](/images/thumb/4/4b/Wiki-create-account.png/250px-Wiki-create-account.png)](//wiki.manjaro.org/index.php?title=File:Wiki-create-account.png)

[](//wiki.manjaro.org/index.php?title=File:Wiki-create-account.png "Enlarge")

creating account link

When Open wiki.manjaro.org, then click Create account link at top-right of the site.The Create Account page should be shown. Fill the fields(your user name should be started with uppercase) and click Create your account button. At least you can see Welcome page.

### [Page Editing]

[![](/images/thumb/e/e1/Searching-page.png/250px-Searching-page.png)](//wiki.manjaro.org/index.php?title=File:Searching-page.png)

[](//wiki.manjaro.org/index.php?title=File:Searching-page.png "Enlarge")

search a page

When you want to edit a page, you should login wiki then find the page with search bar at top-right of the screen.

[![](/images/thumb/3/3a/Edit-a-page.png/250px-Edit-a-page.png)](//wiki.manjaro.org/index.php?title=File:Edit-a-page.png)

[](//wiki.manjaro.org/index.php?title=File:Edit-a-page.png "Enlarge")

Editing a page

Then Click Edit link at top of the page. Before save your changes, please click show preview button to see your changes.

**Info**

------------------------------------------------------------------------

Your changes will send to mods to approve but you can continue to contribute without wait.

### [Adding Page]

[![](/images/thumb/c/ca/Search-for-new-page.png/250px-Search-for-new-page.png)](//wiki.manjaro.org/index.php?title=File:Search-for-new-page.png)

[](//wiki.manjaro.org/index.php?title=File:Search-for-new-page.png "Enlarge")

search for new page

There are many ways to create a page. The easy one is search for page name and click containing\... result.

[![](/images/thumb/2/27/Screenshot_from_2021-09-04_19-53-33.png/250px-Screenshot_from_2021-09-04_19-53-33.png)](//wiki.manjaro.org/index.php?title=File:Screenshot_from_2021-09-04_19-53-33.png)

[](//wiki.manjaro.org/index.php?title=File:Screenshot_from_2021-09-04_19-53-33.png "Enlarge")

new page link

Then Search result should be shown and also Create page link. Click the link then the page creation screen should be shown.

Before save your changes, please click show preview button to see your changes.

\

**Info**

------------------------------------------------------------------------

Your changes will send to mods to approve but you can continue to contribute without wait.
