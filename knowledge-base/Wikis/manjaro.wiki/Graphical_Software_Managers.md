In Manjaro, software management (or package management) is handled by [Pacman](//wiki.manjaro.org/index.php?title=Special:MyLanguage/pacman "Special:MyLanguage/pacman") via command line (terminal). Below are the user-friendly frontend [GUI](https://en.wikipedia.org/wiki/Graphical_user_interface) package managers you can use instead of, or in addition to, using pacman in the terminal. [Pamac](//wiki.manjaro.org/index.php?title=Special:MyLanguage/Pamac "Special:MyLanguage/Pamac") is the default package manager of Manjaro. It can be used via command line ([CLI](https://en.wikipedia.org/wiki/Command-line_interface)) as well.
