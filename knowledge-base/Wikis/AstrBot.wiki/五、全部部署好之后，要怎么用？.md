# GOCQ 
直接在有机器人的地方（单聊、群聊、频道私聊、频道）发送 help 即可打开菜单。

如果 help 中的指令不会用，直接发送那个指令，会有帮助。

下面仅列举几个常用指令

- 使用 nick 可以为其设置唤醒词。
- keyword 设置关键词回复，支持图片
- /revgpt 切换到网页版ChatGPT
- /gpt 切换到官方ChatGPT API
- plugin 插件面板，可以安装插件（插件列表见 [README](https://github.com/Soulter/QQChannelChatGPT)）
- update 热升级机器人
- !myid 查看自己的 id （设置管理者QQ或者频道ID时使用）
- reset 充值绘画
- ...
