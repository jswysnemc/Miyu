# spa-monitor

The PipeWire SPA device debugging utility

# SYNOPSIS

*spa-monitor** *FILE*

# DESCRIPTION

Load a SPA plugin and instantiate a device from it.

This is only useful for debugging device plugins.

# EXAMPLES

*spa-monitor** $(SPA_PLUGINDIR)/jack/libspa-jack.so
