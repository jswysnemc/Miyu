## 15 Errors

Errors returned by the Secret Service API.

## 15.1 `org.freedesktop.Secret.Error.IsLocked`

The object must be unlocked before this action can be carried out.

## 15.2 `org.freedesktop.Secret.Error.NoSession`

The session does not exist.

## 15.3 `org.freedesktop.Secret.Error.NoSuchObject`

No such item or collection exists.
