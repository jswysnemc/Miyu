Build automation (generally referred to as \'Build systems\' in the Gentoo world) is software that automates the compilation, clean up, and installation stages of the software creation process. Recently it has become more common for build automation to perform elements of software testing. Gentoo developers must have at least a general understanding of one or more build systems in order to start writing [ebuilds](https://wiki.gentoo.org/wiki/Ebuild "Ebuild"). This article services as a type of meta article in defining a list of build systems in Gentoo Linux.
