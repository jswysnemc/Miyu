**Extreme Tux Racer** is a popular and very fun arctic racing game forked off of Tux Racer. It includes several different characters and several different tracks, both user-made and also creator-made.

## [Installation]

### [Emerge]

To emerge the package for the game, simply run:

`root `[`#`]`emerge --ask games-action/extreme-tuxracer`

## [Running the game]

If the game cannot be run from a desktop environment or GUI, it is possible to run Extreme Tux Racer from the command line. This is only possible after the package has been successfully installed through portage.

`user `[`$`]`etr`
