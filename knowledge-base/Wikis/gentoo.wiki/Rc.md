This article deals with the history of operating systems. See [OpenRC](https://wiki.gentoo.org/wiki/OpenRC "OpenRC") for the Gentoo init system.

[rc] is an ancient Plan 9 abbreviation that stands for **r**un **c**ommands. Eric Raymond\'s The Jargon File mentions a script of a similar name called rc file. This file contained \"startup instructions for an application program (or an entire operating system)\".^[\[1\]](#cite_note-1)^

It would seem [rc] was the basis for modern [init systems](https://wiki.gentoo.org/wiki/Init_system "Init system").
