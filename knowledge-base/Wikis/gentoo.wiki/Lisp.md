**Lisp** is a family of programming languages related to the LISP (LISt Processor) language designed by John McCarthy at MIT, first described in a 1960 article in the *Communications of the ACM*. Informally, it refers to [Common Lisp](https://en.wikipedia.org/wiki/Common_Lisp "wikipedia:Common Lisp"); however, the Lisp family also includes languages such as [Scheme](https://en.wikipedia.org/wiki/Scheme_(programming_language) "wikipedia:Scheme (programming language)"), described in the R*n*RS series of specifications, together with [Emacs Lisp](https://wiki.gentoo.org/wiki/GNU_Emacs "GNU Emacs") and [Dylan](https://en.wikipedia.org/wiki/Dylan_(programming_language) "wikipedia:Dylan (programming language)").

## [Lisp on Gentoo]

### [Common Lisp]

A number of Common Lisp resources can be found on the [Project:Common Lisp](https://wiki.gentoo.org/wiki/Project:Common_Lisp "Project:Common Lisp") page.

Common Lisp implementations available on Gentoo include:

-   [[[dev-lisp/abcl]](https://packages.gentoo.org/packages/dev-lisp/abcl)[]] - [Armed Bear Common Lisp](//abcl.org), a Common Lisp implementation for the JVM
-   [[[dev-lisp/clisp]](https://packages.gentoo.org/packages/dev-lisp/clisp)[]] - [GNU CLISP](//clisp.sourceforge.io/)
-   [[[dev-lisp/clozurecl]](https://packages.gentoo.org/packages/dev-lisp/clozurecl)[]] - [ClozureCL](//ccl.clozure.com), derived from Digitool\'s MCL product
-   [[[dev-lisp/cmucl]](https://packages.gentoo.org/packages/dev-lisp/cmucl)[]] - [CMU Common Lisp](//www.cons.org/cmucl/)
-   [[[dev-lisp/ecl]](https://packages.gentoo.org/packages/dev-lisp/ecl)[]] - [ECL](//common-lisp.net/project/ecl/), an embeddable Common Lisp implementation
-   [[[dev-lisp/gcl]](https://packages.gentoo.org/packages/dev-lisp/gcl)[]] - [GNU Common Lisp](//www.gnu.org/software/gcl/gcl.html)
-   [[[dev-lisp/sbcl]](https://packages.gentoo.org/packages/dev-lisp/sbcl)[]] - [Steel Bank Common Lisp](//www.sbcl.org/)

There\'s also a virtual package for Common Lisp:

-   [[[virtual/commonlisp]](https://packages.gentoo.org/packages/virtual/commonlisp)[]]

### [Scheme]

A number of Scheme resources can be found on the [Project:Scheme](https://wiki.gentoo.org/wiki/Project:Scheme "Project:Scheme") page.

Scheme implementations available on Gentoo include:

-   [[[dev-scheme/bigloo]](https://packages.gentoo.org/packages/dev-scheme/bigloo)[]] - [Bigloo](//www-sop.inria.fr/indes/fp/Bigloo/index.html), a practical Scheme compiler with many extensions
-   [[[dev-scheme/chez]](https://packages.gentoo.org/packages/dev-scheme/chez)[]] - [Chez](//cisco.github.io/ChezScheme/), a programming language based on R6RS
-   [[[dev-scheme/gambit]](https://packages.gentoo.org/packages/dev-scheme/gambit)[]] - [Gambit](//www.iro.umontreal.ca/~gambit/), a native Scheme to C compiler and interpreter
-   [[[dev-scheme/gauche]](https://packages.gentoo.org/packages/dev-scheme/gauche)[]] - [Gauche](//practical-scheme.net/gauche/), a Unix system friendly Scheme interpreter
-   [[[dev-scheme/guile]](https://packages.gentoo.org/packages/dev-scheme/guile)[]] - [GNU Ubiquitous Intelligent Language for Extensions](//www.gnu.org/software/guile/)
-   [[[dev-scheme/kawa]](https://packages.gentoo.org/packages/dev-scheme/kawa)[]] - [Kawa](//www.gnu.org/software/kawa/), a Java-based Scheme system and Language Framework
-   [[[dev-scheme/racket]](https://packages.gentoo.org/packages/dev-scheme/racket)[]] - [Racket](https://wiki.gentoo.org/wiki/Racket "Racket"), a general purpose, multi-paradigm Lisp-Scheme programming language
