**Resources**

[[]][Wikipedia](https://en.wikipedia.org/wiki/File_manager "wikipedia:File manager")

This article provides a list of file managers available in Gentoo.

A **file manager** is a computer program that allows for the manipulation of files and directories on a computer\'s [filesystem](https://wiki.gentoo.org/wiki/Filesystem "Filesystem").

### [Command line]

Command line file managers are usually run in a [terminal emulator](https://wiki.gentoo.org/wiki/Terminal_emulator "Terminal emulator"), and often configured through configuration files. They can generally run remotely over [SSH](https://wiki.gentoo.org/wiki/SSH "SSH").
