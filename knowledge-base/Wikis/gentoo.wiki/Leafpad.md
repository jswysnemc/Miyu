Leafpad is an extremely simple, yet functional, GTK2 text editor. It is the default text editor used by [LXDE](https://wiki.gentoo.org/wiki/LXDE "LXDE").^[\[1\]](#cite_note-1)^

## [Features]

Some of Leafpad\'s features^[\[2\]](#cite_note-2)^ include:

-   Codeset option (Some OpenI18N registered)
-   Auto codeset detection (UTF-8 and some codesets)
-   Unlimitted Undo/Redo
-   Auto/Multi-line Indent
-   Display line numbers
-   Drag and Drop
-   Printing
