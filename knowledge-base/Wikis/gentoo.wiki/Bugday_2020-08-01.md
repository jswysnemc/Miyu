## [Topic 1: Old bugs]

Search old bug tickets and check if it is

-   solved already
-   still reproducible with the latest version
-   is there a fix upstream already?
-   are there other important news about it?

### [Resources]

The Bug Cleaners\' list of [resources](https://wiki.gentoo.org/wiki/Project:Bug-cleaners#Resources "Project:Bug-cleaners") is a good place to start.

Bugs assigned to obsolete projects in Gentoo: [link](https://bugs.gentoo.org/buglist.cgi?email1=disabled&emailassigned_to1=1&emailtype1=substring&list_id=4730146&query_format=advanced&resolution=---)
