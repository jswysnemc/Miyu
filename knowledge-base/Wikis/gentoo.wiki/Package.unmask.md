**[/etc/portage/package.unmask]** is a file containing a list of packages that are to be unmasked. It is similar to the [[package.mask]](https://wiki.gentoo.org/wiki//etc/portage/package.mask "/etc/portage/package.mask") file and useful for overriding values set in Portage\'s global [package.mask] file.

** Note**\
This file does not override packages that are masked via [`KEYWORDS`](https://wiki.gentoo.org/wiki/KEYWORDS "KEYWORDS").
