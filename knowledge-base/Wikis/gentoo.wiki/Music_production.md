## [External resources]

-   [Linux DAW](https://linuxdaw.org/) --- Lists software for Linux Digital Audio Workstations
-   [LV2](https://en.wikipedia.org/wiki/LV2 "wikipedia:LV2") - LADSPA Version 2, successor to LADSPA and replacement for DSSI.
-   [DSSI](https://en.wikipedia.org/wiki/Disposable_Soft_Synth_Interface "wikipedia:Disposable Soft Synth Interface") - Disposable Soft Synth Interface, virtual instrument plugin architecture built on LADSPA.
-   [LADSPA](https://en.wikipedia.org/wiki/LADSPA "wikipedia:LADSPA") - Linux Audio Developer\'s Simple Plugin API, standard for handling audio filters and audio signal processing effects.
