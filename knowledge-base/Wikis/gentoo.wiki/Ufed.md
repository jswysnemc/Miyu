**ufed** is an ncurses-based USE flag editor designed to simplify configuration of the USE flags set in [make.conf](https://wiki.gentoo.org/wiki//etc/portage/make.conf "/etc/portage/make.conf").

** Note**\
ufed does not allow setting flags per-package, which is done with [/etc/portage/package.use](https://wiki.gentoo.org/wiki//etc/portage/package.use "/etc/portage/package.use").

## [Usage]

[![](/images/thumb/8/8e/Ufed-0.92_screenshot.png/200px-Ufed-0.92_screenshot.png)](https://wiki.gentoo.org/wiki/File:Ufed-0.92_screenshot.png)

[](https://wiki.gentoo.org/wiki/File:Ufed-0.92_screenshot.png "Enlarge")

Screenshot of [[[app-portage/ufed]](https://packages.gentoo.org/packages/app-portage/ufed)[]]-0.92

Start the program from a command-line with the [ufed] command. It may take a few seconds for the program to open; do not be alarmed if it seems like nothing is happening, just wait a little longer.

`root `[`#`]`ufed`

** Note**\
Since [ufed] has the ability to modify important system files, it can only be started using the equivalent of root privileges. Normal users can use [[sudo](https://wiki.gentoo.org/wiki/Sudo "Sudo")] if it is installed on the system, otherwise access to the root account is needed.
