## [Configuration]

Chipset: **Realtek RTL8761B**, so the following needs to be enabled for [[[sys-kernel/linux-firmware]](https://packages.gentoo.org/packages/sys-kernel/linux-firmware)[]]:

[FILE] **`/etc/portage/savedconfig/sys-kernel/linux-firmware`**

    rtl_bt/rtl8761bu_fw.bin
    rtl_bt/rtl8761bu_config.bin

For instructions on use, see [Device pairing](https://wiki.gentoo.org/wiki/Bluetooth#Device_pairing "Bluetooth").
