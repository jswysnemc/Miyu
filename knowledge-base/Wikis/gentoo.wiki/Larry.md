## [Appearances]

[![SVG graphic of Larry the cow](/images/thumb/3/3d/Larry_color.svg/300px-Larry_color.svg.png)](https://wiki.gentoo.org/wiki/File:Larry_color.svg)

[](https://wiki.gentoo.org/wiki/File:Larry_color.svg "Enlarge")

Larry after he grew his black and white patches.

Here on the wiki the occasional use of Larry\'s name can be used as a substitute when needed.

Using Larry\'s name for this purpose is beneficial for a few reasons:

1.  It helps establish a *consistent style* for readers and editors to keep in mind.
2.  Using Larry\'s name in appropriate places helps the Wiki\'s documentation feel more fun and coherent. Many readers enjoy humor and [Easter eggs](https://en.wikipedia.org/wiki/Easter_egg_(media) "wikipedia:Easter egg (media)").
3.  It helps Mr. Larry feel more included. Instead of feeling lonely, like he has been off in the pasture all by himself *for days*.

For example, the [gpasswd] command could be used with Larry\'s name and an example group instead of simply `<user>` and `<group>`:

`root `[`#`]`gpasswd -a larry <group>`

As opposed to:

`root `[`#`]`gpasswd -a <user> <group>`

Both could work. One method is not better than the other. But the first method gets more style points.

## [Personal life]

In the past there have been discussions over Larry the Cow\'s name. Since Larry is a male (he has no udders), some have argued that he might more appropriately be called \"Larry the Bull\". A [[[bug was filed]](https://bugs.gentoo.org/show_bug.cgi?id=27727)[]] discussing this, and it was decided to take no action - so \"Larry the Cow\" he is!
