Gentoo NetBSD (or Gentoo/NBSD, or G/NBSD) was an effort to create a complete [NetBSD-based](http://netbsd.org/) Gentoo system, sharing the complete administration facilities of Gentoo with the reliability of the NetBSD kernel and userland.

It offered a stage3 for Gentoo NetBSD, so installation is possible without the need for a regular NetBSD installation. To install Gentoo NetBSD an [install guide](https://wiki.gentoo.org/wiki/Gentoo_NetBSD/Install_Guide "Gentoo NetBSD/Install Guide") was written.

This project is now defunct.
