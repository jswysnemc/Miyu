[[]][Wikipedia](https://en.wikipedia.org/wiki/Software_configuration_management "wikipedia:Software configuration management")

Configuration management software provides facilities for managing software configuration, in a centralized or hierarchical manner, with change monitoring.

## [Available software]

This is a partial selection of configuration management tools available in Gentoo. See packages.gentoo.org, or use [eix](https://wiki.gentoo.org/wiki/Eix "Eix") ([[[app-portage/eix]](https://packages.gentoo.org/packages/app-portage/eix)[]]) for more packages.

  ----------------------------------------------------------- --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- ----------------------------------------------------------------------------------
  Name                                                        Package                                                                                                                                                                                                                                                                                                                                                                     Description
  [Ansible](https://wiki.gentoo.org/wiki/Ansible "Ansible")   [[[app-admin/ansible]](https://packages.gentoo.org/packages/app-admin/ansible)[]]   Model-driven deployment, config management, and command execution framework.
  cdist                                                       [[[app-admin/cdist]](https://packages.gentoo.org/packages/app-admin/cdist)[]]         Usable configuration management system.
  cfengine                                                    [[[net-misc/cfengine]](https://packages.gentoo.org/packages/net-misc/cfengine)[]]   Automated suite of programs for configuring and maintaining Unix-like computers.
  [Puppet](https://wiki.gentoo.org/wiki/Puppet "Puppet")      [[[app-admin/puppet]](https://packages.gentoo.org/packages/app-admin/puppet)[]]      System automation and configuration management software.
  Salt                                                        [[[app-admin/salt]](https://packages.gentoo.org/packages/app-admin/salt)[]]            Remote execution and configuration manager.

## [External references]

-   [[[dev-ruby/capistrano]](https://packages.gentoo.org/packages/dev-ruby/capistrano)[]] - A distributed application deployment system.
-   [[[app-admin/consul]](https://packages.gentoo.org/packages/app-admin/consul)[]] - A tool for service discovery, monitoring and configuration.
-   [Elektra](https://www.libelektra.org/) - Stores configuration in a global, hierarchical key database. Not currently in the Gentoo repository.
-   [[[dev-db/etcd]](https://packages.gentoo.org/packages/dev-db/etcd)[]] \-- Highly-available key value store for shared configuration and service discovery.
