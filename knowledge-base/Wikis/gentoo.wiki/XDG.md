[XDG] is an abbreviation for the X Desktop Group, now known as [freedesktop.org](https://freedesktop.org). freedesktop.org creates and maintains specifications and software related to desktop systems.

For information about:

-   The XDG Base Directory specification, which describes environment variables such as `XDG_RUNTIME_DIR` and their default values, refer to the [XDG/Base Directories](https://wiki.gentoo.org/wiki/XDG/Base_Directories "XDG/Base Directories") page.

<!-- -->

-   The XDG User Directories specification, which can be used to specify which directories should be used for \"well-known\" directories such as for music and pictures, refer to the [XDG/User Directories](https://wiki.gentoo.org/wiki/XDG/User_Directories "XDG/User Directories") page.

<!-- -->

-   The [xdg-desktop-portal] interface, which provides a standard API for things such as file access, opening URIs, and printing, refer to the [XDG/xdg-desktop-portal](https://wiki.gentoo.org/wiki/XDG/xdg-desktop-portal "XDG/xdg-desktop-portal") page.

<!-- -->

-   Command-line programs for managing and using default applications for particular [MIME](https://en.wikipedia.org/wiki/MIME "wikipedia:MIME") types, such as `text/plain`, `image/gif` and `audio/mp3`, refer to the [XDG/Software](https://wiki.gentoo.org/wiki/XDG/Software "XDG/Software") page.
