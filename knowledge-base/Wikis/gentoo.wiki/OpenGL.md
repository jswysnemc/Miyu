**OpenGL** is a graphics API created by The Khronos Group. It\'s designed to primarily to be used on desktop systems, alongside counterparts such as [OpenGL ES](https://www.khronos.org/opengles/) targeting mobile devices or [WebGL](https://www.khronos.org/webgl/) for websites.

## [Installation]

Drivers are included in [[[media-libs/mesa]](https://packages.gentoo.org/packages/media-libs/mesa)[]] and cannot be removed since it\'s integral to the system.

-   [OpenCL](https://wiki.gentoo.org/wiki/OpenCL "OpenCL") --- a framework for writing programs that execute across heterogeneous computing platforms (CPUs, GPUs, DSPs, FPGAs, ASICs, etc.).
