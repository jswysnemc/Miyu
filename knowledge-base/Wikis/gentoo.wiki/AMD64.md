The **[amd64]** is a 64-bit architecture officially supported within Gentoo through the [AMD64 Project](https://wiki.gentoo.org/wiki/Project:AMD64 "Project:AMD64").

This architecture is compatible with the traditional x86 architecture (and thus also known as *x86_64*). It was developed by AMD as *AMD64* and later adopted by Intel as *EM64T*. As of 2020, it is the most widely used architecture for PCs, notebooks, and server systems.

The AMD64 processor family includes:

-   AMD Athlon 64/Opteron
-   AMD Phenom
-   AMD FX
-   AMD Ryzen
-   Intel Core 2
-   Intel Core i3/i5/i7/i9
