**tldr pages** (\"too long didn\'t read\") is a project to provide brief documentation for [CLI](https://wiki.gentoo.org/wiki/Shell "Shell") commands.

Complementary to [man pages](https://wiki.gentoo.org/wiki/Man_page "Man page") and [Info](https://wiki.gentoo.org/wiki/Info "Info") documentation, tldr pages provide a quick reference with examples. They can be a handy way to jog memory about basic command functionality, but don\'t aim to provide comprehensive documentation.

## [Available software]

[Ebuilds](https://wiki.gentoo.org/wiki/Ebuild "Ebuild") for two *tldr pages* clients are currently available in the [GURU](https://wiki.gentoo.org/wiki/Project:GURU "Project:GURU") repository. See the [GURU guide](https://wiki.gentoo.org/wiki/Project:GURU/Information_for_End_Users "Project:GURU/Information for End Users") and [emerge](https://wiki.gentoo.org/wiki/Emerge "Emerge") for installation instructions.
