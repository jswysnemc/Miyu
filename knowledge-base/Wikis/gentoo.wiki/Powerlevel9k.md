**Powerlevel9k** is a theme for [ZSH](https://wiki.gentoo.org/wiki/Zsh "Zsh") which uses Powerline Fonts. It can be used with vanilla ZSH or ZSH frameworks such as Oh-My-Zsh, Prezto, Antigen, and many others.

## [Installation]

### [Prerequisites]

The Powerlevel9k theme requires [[[app-shells/zsh]](https://packages.gentoo.org/packages/app-shells/zsh)[]].

## [Configuration]

After configuring zsh correctly, add this command at the begin of the [\~/.zshrc] file to source the Powerlevel9k theme:

[FILE] **`~/.zshrc`**

    source "/usr/share/zsh-theme-powerlevel9k/powerlevel9k.zsh-theme"

See git Powerlevel9k repository to configure more this theme: [https://github.com/Powerlevel9k/powerlevel9k](https://github.com/Powerlevel9k/powerlevel9k).
