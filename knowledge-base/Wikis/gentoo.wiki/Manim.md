## [Installation]

** Important**\
Manim requires [[[media-video/ffmpeg]](https://packages.gentoo.org/packages/media-video/ffmpeg)[]] to have certain codec USE flag like `x264` to be enabled.

Manim can be installed via [pip](https://wiki.gentoo.org/wiki/Pip#Package_installation "Pip").

### [LaTex]

To use LaTex, install [[[app-text/texlive]](https://packages.gentoo.org/packages/app-text/texlive)[]] with `extra` and `science` USE flag.

## [Usage]

There is a [quickstart](https://docs.manim.community/en/stable/tutorials/quickstart.html) tutorial to get beginners started. To play videos/images, browse through the [media] directory and play with a video player or web browser.
