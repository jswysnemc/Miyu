**System** could have a variety of meanings:

-   [systemd](https://wiki.gentoo.org/wiki/Systemd "Systemd") --- a modern SysV-style init and [[rc](https://wiki.gentoo.org/wiki/Rc "Rc")] replacement for Linux systems.
-   [System set (Portage)](https://wiki.gentoo.org/wiki/System_set_(Portage) "System set (Portage)") --- the software packages required for a standard Gentoo Linux installation to run properly.
-   [System time](https://wiki.gentoo.org/wiki/System_time "System time") --- is used in Unix systems to keep track of time.
