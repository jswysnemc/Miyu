E**x**tensible **M**essaging and **P**resence **P**rotocol (XMPP), formerly known as Jabber, is a real-time communication protocol based on XML. The protocol is an open standard and allows federated messaging between different servers and clients. XMPP is extensible and in active development.

Some examples of specified features include:

-   Direct and group chat messages.
-   Encryption (via OMEMO or OpenPGP).
-   File transfer.
-   Audio/video calls.
-   Typing notifications.
-   Message receipts.
-   Message correction.
