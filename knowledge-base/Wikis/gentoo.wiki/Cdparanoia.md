**cdparanoia** is a tool for ripping CDs. The \"paranoia\" in **cdparanoia** refers to its data verification features.

## [Installation]

cdparanoia is available in the Gentoo repositories and can be installed with the following command:

`root `[`#`]`emerge --ask media-sound/cdparanoia`

For managing the symlinks of the two different, currently available cdparanoia packages eselect can be used with the following package: [[[app-eselect/eselect-cdparanoia]](https://packages.gentoo.org/packages/app-eselect/eselect-cdparanoia)[]].

## [Usage]

Assuming your CD is located at [/dev/sr0] or [/dev/cdrom], run cdparanoia with the following flags to extract all audio present on the CD in WAV format into the current working directory. The tracks are split at the track boundary, making them separate files:

`user `[`$`]`cdparanoia --batch --output-wav`

Or alternatively, and shorter:

`user `[`$`]`cdparanoia -Bw`

The files generated by the above command can be converted to [FLAC] or [MP3] with either [[[media-libs/flac]](https://packages.gentoo.org/packages/media-libs/flac)[]] or [[[media-sound/lame]](https://packages.gentoo.org/packages/media-sound/lame)[]] respectively.
