  [[lsusb](https://wiki.gentoo.org/wiki/Usbutils "Usbutils")]                              [[[sys-apps/usbutils]](https://packages.gentoo.org/packages/sys-apps/usbutils)[]]            Includes command-line tools for identifying USB system hardware.
