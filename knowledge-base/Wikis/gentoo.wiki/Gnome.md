**GNOME** is a feature-rich desktop environment provided by the [GNOME project](https://www.gnome.org). It is one of the most widely-used desktop environments for Linux desktops and has been an important part of Gentoo for many years.

Gentoo currently has GNOME version 40 and beyond, that features yet another redesigned UI optimized for simplicity and gesture usage. Getting familiar with it only takes a matter of minutes. A rather good starting point to quickly get familiar with this new UI is the [GNOME Fourty](https://forty.gnome.org/) page.

** Important**\
Attempting to launch GNOME on Wayland while using the proprietary (binary blob) NVIDIA kernel module is unsupported. See [this troubleshooting section](https://wiki.gentoo.org/wiki/GNOME/GDM#GDM_crashes_when_attempting_to_launch_a_GNOME_Wayland_session "GNOME/GDM") for more details.

The **recommended** way to setup GNOME on Gentoo is described in the [GNOME Guide](https://wiki.gentoo.org/wiki/GNOME/Guide "GNOME/Guide") sub-article for a more detailed approach on installing, configuring, and using GNOME.
