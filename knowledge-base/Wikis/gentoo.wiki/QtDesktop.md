### [Text and office apps]

-   [[[app-editors/focuswriter]](https://packages.gentoo.org/packages/app-editors/focuswriter)[]] - fullscreen and distraction-free word processor
-   [[[app-editors/juffed]](https://packages.gentoo.org/packages/app-editors/juffed)[]] - tabbed text editor with syntax highlighting
-   [[[app-editors/tea]](https://packages.gentoo.org/packages/app-editors/tea)[]] - small, lightweight text editor
-   [[[app-editors/qxmledit]](https://packages.gentoo.org/packages/app-editors/qxmledit)[]] - Qt XML editor
-   [[[app-office/lyx]](https://packages.gentoo.org/packages/app-office/lyx)[]] - frontend for LaTeX, DocBook, etc.
-   [[[app-office/scribus]](https://packages.gentoo.org/packages/app-office/scribus)[]] - desktop publishing and layout program
-   [[[app-office/texmaker]](https://packages.gentoo.org/packages/app-office/texmaker)[]] - frontend for LaTeX
-   [[[app-office/texstudio]](https://packages.gentoo.org/packages/app-office/texstudio)[]] - Free cross-platform LaTeX editor (fork from texmakerX)
-   [[[app-text/calibre]](https://packages.gentoo.org/packages/app-text/calibre)[]] - ebook management
-   [[[app-text/kchmviewer]](https://packages.gentoo.org/packages/app-text/kchmviewer)[]] - feature rich chm file viewer
-   [[[app-text/qpdfview]](https://packages.gentoo.org/packages/app-text/qpdfview)[]] - tabbed PDF viewer

### [Multimedia]

-   [[[media-gfx/freecad]](https://packages.gentoo.org/packages/media-gfx/freecad)[]] - Parametric CAD application
-   [[[media-gfx/nomacs]](https://packages.gentoo.org/packages/media-gfx/nomacs)[]] - image viewer
-   [[[media-gfx/photoqt]](https://packages.gentoo.org/packages/media-gfx/photoqt)[]] - Simple but powerful Qt-based image viewer
-   [[[media-gfx/phototonic]](https://packages.gentoo.org/packages/media-gfx/phototonic)[]] - image viewer
-   [[[media-sound/audacious]](https://packages.gentoo.org/packages/media-sound/audacious)[]] - audio player
-   [[[media-sound/bempc]](https://packages.gentoo.org/packages/media-sound/bempc)[]] - Music Player Daemon client with experimental UI
-   [[[media-sound/cantata]](https://packages.gentoo.org/packages/media-sound/cantata)[]] - featureful Music Player Daemon client
-   [[[media-sound/clementine]](https://packages.gentoo.org/packages/media-sound/clementine)[]] - modern music player and library organizer
-   [[[media-sound/strawberry]](https://packages.gentoo.org/packages/media-sound/strawberry)[]] - modern music player and library organizer based on Clementine and Qt
-   [[[media-sound/coquillo]](https://packages.gentoo.org/packages/media-sound/coquillo)[]] - audio tagger
-   [[[media-sound/flacon]](https://packages.gentoo.org/packages/media-sound/flacon)[]] - extracts audio tracks from audio CD image to separate tracks
-   [[[media-sound/qastools]](https://packages.gentoo.org/packages/media-sound/qastools)[]] - ALSA mixer GUI
-   [[[media-sound/qmmp]](https://packages.gentoo.org/packages/media-sound/qmmp)[]] - Audio player with Winamp/XMMS skins support
-   [[[media-sound/quimup]](https://packages.gentoo.org/packages/media-sound/quimup)[]] - Music Player Daemon client with cover art display
-   [[[media-video/avidemux]](https://packages.gentoo.org/packages/media-video/avidemux)[]] - video editor/transcoder
-   [[[media-video/baka-mplayer]](https://packages.gentoo.org/packages/media-video/baka-mplayer)[]] - libmpv-based media player
-   [[[media-video/smplayer]](https://packages.gentoo.org/packages/media-video/smplayer)[]] - great MPlayer/mpv frontend
-   [[[media-video/vlc]](https://packages.gentoo.org/packages/media-video/vlc)[]] - media player

### [Network apps]

-   [[[net-dialup/cutecom]](https://packages.gentoo.org/packages/net-dialup/cutecom)[]] - serial terminal
-   [[[net-im/psi]](https://packages.gentoo.org/packages/net-im/psi)[]] - instant messenger
-   [[[net-im/vacuum]](https://packages.gentoo.org/packages/net-im/vacuum)[]] - Jabber client
-   [[[net-irc/kvirc]](https://packages.gentoo.org/packages/net-irc/kvirc)[]] - Advanced IRC Client
-   [[[net-misc/fatrat]](https://packages.gentoo.org/packages/net-misc/fatrat)[]] - download/upload manager
-   [[[net-news/rssguard]](https://packages.gentoo.org/packages/net-news/rssguard)[]] - RSS/Atom feed reader
-   [[[net-p2p/qbittorrent]](https://packages.gentoo.org/packages/net-p2p/qbittorrent)[]] - BitTorrent client
-   [[[net-p2p/transmission]](https://packages.gentoo.org/packages/net-p2p/transmission)[]] - BitTorrent client
-   [[[net-wireless/wpa_supplicant]](https://packages.gentoo.org/packages/net-wireless/wpa_supplicant)[]] - Wi-Fi front and back end (see also [wpa_supplicant](https://wiki.gentoo.org/wiki/Wpa_supplicant "Wpa supplicant"))
-   [[[www-client/falkon]](https://packages.gentoo.org/packages/www-client/falkon)[]] - Web browser
-   [[[www-client/otter]](https://packages.gentoo.org/packages/www-client/otter)[]] - Web browser
-   [[[www-client/qutebrowser]](https://packages.gentoo.org/packages/www-client/qutebrowser)[]] - Web browser

### [Desktop utilities]

-   [[[app-admin/keepassxc]](https://packages.gentoo.org/packages/app-admin/keepassxc)[]] - password manager
-   [[[sci-calculators/speedcrunch]](https://packages.gentoo.org/packages/sci-calculators/speedcrunch)[]] - calculator
-   [[[sys-apps/qdirstat]](https://packages.gentoo.org/packages/sys-apps/qdirstat)[]] - Qt application analyzing disk space allocation
-   [[[x11-apps/copyq]](https://packages.gentoo.org/packages/x11-apps/copyq)[]] - clipboard manager with advanced features
-   [[[x11-misc/qlipper]](https://packages.gentoo.org/packages/x11-misc/qlipper)[]] - clipboard history applet
-   [[[x11-misc/qps]](https://packages.gentoo.org/packages/x11-misc/qps)[]] - process manager
-   [[[x11-misc/qxkb]](https://packages.gentoo.org/packages/x11-misc/qxkb)[]] - keyboard layout switcher
-   [[[x11-terms/qterminal]](https://packages.gentoo.org/packages/x11-terms/qterminal)[]] - multitab terminal emulator
-   [[[x11-misc/qtfm]](https://packages.gentoo.org/packages/x11-misc/qtfm)[]] - lightweight file manager
