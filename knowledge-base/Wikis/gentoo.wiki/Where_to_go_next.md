** Important**\
This a beta project to extend the Handbook so users aren\'t left to fend for themselves after installing Gentoo. The idea is to become a table of contents for needs a user might have without being overloaded.

** Important**\
Additions are welcome, however please bare in mind that Gentoo is about user choice so we shouldn\'t be picking the software for them and rather guiding them to the popular choices.

So Gentoo is installed, but what is next?

The where to go next project is an effort to centralize a directory of all the common usages a Gentoo Linux user may wish to configure a system for their exact needs. Be that from setting up a desktop environment, installing a preferred web browser or even setting up a LAMP server. The answer should be here!

Due to the many options that portage makes available to it\'s users, it can be extremely overwhelming trying to learn them all once. To make it more manageable the wiki docs in Gentoo Linux are designed to teach these features as a user needs them. Some articles do brush over some of these of terms as they are deemed common knowledge learnt in a handbook install. Because of that, there is brief overview of some of most used portage config files and system admin tools in the first section, just so users don\'t see those articles as black magic.

The guide is split in to three categories:

[Maintaining a Gentoo Linux system](https://wiki.gentoo.org/wiki/Where_to_go_next/Maintain "Where to go next/Maintain")

[Desktop Systems](https://wiki.gentoo.org/wiki/Where_to_go_next/Desktop "Where to go next/Desktop")
