The purpose of this page is to collect information on all current distributions that are based on Gentoo.

Users are encouraged to extend and update the list below. Before adding a new entry, **please verify** that the distribution in question is actually based on Gentoo (i.e. built using [Ebuilds](https://wiki.gentoo.org/wiki/Ebuild "Ebuild")). To qualify as current, the latest release of release-based distributions should be **no older than 12 months**, project\'s websites need to be accessible, at least. Thanks!

** Warning**\
These distributions are neither tested not endorsed by the documentation project. Users should make their own judgements as to whether they are stable, secure, and have sufficient development or support.
