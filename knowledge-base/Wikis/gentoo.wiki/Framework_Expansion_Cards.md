The [Framework Laptop 13](https://wiki.gentoo.org/wiki/Framework_Laptop_13 "Framework Laptop 13") and [Framework Laptop 16](https://wiki.gentoo.org/wiki/Framework_Laptop_16 "Framework Laptop 16") use an expansion card system that allows the user to customize the laptop\'s ports and functionality. Each connects to an underlying USB-C interface. See the [Framework marketplace](https://frame.work/marketplace/expansion-cards).
