**ChromeOS** is an operating system developed almost exclusively by Google in order to be used on the Chromebook hardware line.

ChromeOS is built^[\[1\]](#cite_note-1)^ using [Portage](https://wiki.gentoo.org/wiki/Portage "Portage")^[\[2\]](#cite_note-2)^, Gentoo\'s package manager, and Gentoo-based [chroots](https://wiki.gentoo.org/wiki/Chroot "Chroot")^[\[3\]](#cite_note-3)^. ChromeOS uses the [upstart](https://en.wikipedia.org/wiki/Upstart_(software) "wikipedia:Upstart (software)") [init system](https://wiki.gentoo.org/wiki/Init_system "Init system").

[ChromeOS Flex](https://chromeenterprise.google/os/chromeosflex/) now allows ChromeOS to be installed on many commodity machines.
