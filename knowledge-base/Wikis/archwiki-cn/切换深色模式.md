在明亮模式和黑暗模式/主题之间切换非常有趣。它允许您在日落时切换到黑暗模式，或使用键盘快捷键切换模式。 

主题之间的切换可以应用于当前正在运行的程序（可能需要守护程序），或者仅应用于新启动的程序。本文重点介绍在运行时进行切换，因此在使用过程中进行切换会影响当前正在运行的程序。 

在明暗模式之间切换需要来自应用程序或应用程序工具包（例如 [GTK](<../zh-cn/GTK.html> "GTK") 和 [Qt](<../zh-cn/Qt.html> "Qt")）的支持。 

若你只是想要快速简便地将整个显示风格调暗，可以尝试以下操作： [安装](<../zh-cn/Help:%E9%98%85%E8%AF%BB.html#%E5%AE%89%E8%A3%85%E8%BD%AF%E4%BB%B6%E5%8C%85> "安装") [gnome-themes-extra](<https://archlinux.org/packages/?name=gnome-themes-extra>)包， [gnome-themes-extra-gtk2](<https://aur.archlinux.org/packages/gnome-themes-extra-gtk2/>)AUR， [adwaita-qt5-git](<https://aur.archlinux.org/packages/adwaita-qt5-git/>)AUR [adwaita-qt6-git](<https://aur.archlinux.org/packages/adwaita-qt6-git/>)AUR 以及 [qt6ct](<https://archlinux.org/packages/?name=qt6ct>)包。在 [qt6ct](<https://archlinux.org/packages/?name=qt6ct>)包 中选择主题为 Adwaita-Dark，并且设置后面这些[环境变量](<../zh-cn/%E7%8E%AF%E5%A2%83%E5%8F%98%E9%87%8F.html> "环境变量")：`GTK_THEME=Adwaita:dark`， `GTK2_RC_FILES=/usr/share/themes/Adwaita-dark/gtk-2.0/gtkrc` 以及 `QT_QPA_PLATFORMTHEME=qt5ct`。最后，在“终端”输入这行命令：`gsettings set org.gnome.desktop.interface gtk-theme Adwaita-dark`

##  工具包

### GTK

要更改明/暗模式，您必须更改使用的主题。 

大多数主题的确有深色变体，按照惯例，后缀为 `-dark`。例如，默认的 [GTK](<../zh-cn/GTK.html> "GTK") 主题 `Adwaita` 具有变体 `Adwaita-dark`。 

要永久更改为深色变体，请参见 [GTK#深色版主题](<../zh-cn/GTK.html#%E6%B7%B1%E8%89%B2%E7%89%88%E4%B8%BB%E9%A2%98> "GTK")。 

要立即切换主题以运行程序，需要提供 [xsettings 规范](<https://www.freedesktop.org/wiki/Specifications/xsettings-spec/>)的守护程序。 

[Xfce](<../zh-cn/Xfce.html> "Xfce") 中的一个是 [xfsettingsd](<https://docs.xfce.org/xfce/xfce4-settings/xfsettingsd>)，由 [xfce4-settings](<https://archlinux.org/packages/?name=xfce4-settings>)包 包提供。 

要查询当前的 GTK 主题： 
    
    $ xfconf-query -c xsettings -p /Net/ThemeName
    
设置 GTK 主题： 
    
    $ xfconf-query -c xsettings -p /Net/ThemeName -s "new-theme"
    
对该条目的更改是即时的，并且会影响所有 GTK 应用程序。 

### Qt

[Qt](<../zh-cn/Qt.html> "Qt") 具有类似于 GTK 的主题支持。 

Qt 应用程序主题化的一种方法是[使用 GTK 进行样式设置](<../zh-cn/Uniform_look_for_Qt_and_GTK_applications.html#QGtkStyle> "Uniform look for Qt and GTK applications")。GTK 主题的更改也会影响 Qt 应用程序。 

##  应用程序

### Firefox

[Firefox](<../zh-cn/Firefox.html> "Firefox") 自动使用当前的 GTK 主题模式，并相应地调整浏览器的外观。有关更多设置和注意事项，请参见 [Firefox#Dark themes](<../zh-cn/Firefox.html#Dark_themes> "Firefox")。 

要聪明地更改 Web 内容，建议使用 [Dark Reader](<https://addons.mozilla.org/firefox/addon/darkreader/>) 附加组件。 

通过将 `Automation` 设定为 `Use system color scheme`，深色阅读器会自动使用深色 GTK 主题激活。 

### Thunderbird

[Thunderbird](<../zh-cn/Thunderbird.html> "Thunderbird") 符合当前的 GTK 主题，但建议进行一些更改。 

请参阅 [Thunderbird#Theming tweaks](<../zh-cn/Thunderbird.html#Theming_tweaks> "Thunderbird")。 

### Visual Studio Code

要在 [Visual Studio Code](<../zh-cn/Visual_Studio_Code.html> "Visual Studio Code") 中更改主题，[此脚本](<https://github.com/sandygk/woof-configs/blob/master/bin/set_theme_vscode>)可能会有所帮助。 

### Alacritty

[Alacritty](<../zh-cn/Alacritty.html> "Alacritty") 支持多种自定义配色方案。可以在[此处](<https://github.com/alacritty/alacritty/wiki/Color-schemes>)找到配置语法和已发布的配色方案。 

要快速更改主题，应声明一个指向每种配色方案的指针，例如 `&black`。然后，您只需设置 `colors: *black` 就可以切换到配色方案。对配置文件的更改是即时的，并且会影响所有当前正在运行的程序。 

边框和标题栏以 GTK 为主题。为了遵守 GTK 主题，您应该将 `gtk_theme_variant` 设置为默认值 `None`。
