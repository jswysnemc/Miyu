来自[使用 TeX, LaTeX 及类似程序](<https://www.tug.org/begin.html>): 

     [TeX](<https://www.tug.org/>) 是类型设置语言，它并不设置文本的显示格式，而是将文本与 Tex 命令结合到一个纯文本文件中。运行 Tex 就可以将文件转化成格式化的文本，例如 PDF 文件。所以，与标准文字处理器不同，您的文档是个单独的文件，并不是最终要展示的结果，非常易于编辑和处理。

TeX 仅是 _TeX 发布版本_ 的一个组件。除它外，还有提供了许多宏软件包(也被称为 TeX 格式)的[LaTeX](<../zh-cn/TeX_Live.html> "LaTeX")和非常简化的 Plain TeX。 Tex 在内部使用 Metafont 定义字形和直接的连接关系。 

[TeX Live](<../zh-cn/TeX_Live.html> "TeX Live") 是一个完整、功能强大的 TeX 发布版本，包含了主要的 Tex 相关程序、宏和字体，[官方软件仓库](<../zh-cn/%E5%AE%98%E6%96%B9%E4%BB%93%E5%BA%93.html> "官方仓库")收录了它。 老的(停止开发) teTeX 发布版本位于 [AUR](<../zh-cn/Arch_%E7%94%A8%E6%88%B7%E8%BD%AF%E4%BB%B6%E4%BB%93%E5%BA%93.html> "AUR")。
