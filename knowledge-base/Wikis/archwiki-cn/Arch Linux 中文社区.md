##  即时聊天群组

Arch Linux 中文社区在不同的平台均有数个即时聊天群组，其中交流 Arch Linux 相关信息的群组被称为“主群”，而闲聊话题单独前往一个群组，被称为“ot群”（ot 是 off-topic 的缩写）。不同的接入方式是由机器人互通的。历史上 IRC 用户曾经占多数，而现今绝大多数用户位于 Telegram 或 Matrix 上。另请参阅 [Project:Arch Linux 中文社区交流群指引](<../Project:Arch_Linux_%E4%B8%AD%E6%96%87%E7%A4%BE%E5%8C%BA%E4%BA%A4%E6%B5%81%E7%BE%A4%E6%8C%87%E5%BC%95.html> "Project:Arch Linux 中文社区交流群指引")。 

###  [Telegram](<../zh-cn/Telegram.html> "Telegram") 群组

因我们的反垃圾机制的要求，请先关注我们的频道 <https://t.me/archlinuxcn>，然后访问 <https://t.me/archlinuxcn_group> 加入群组。成功加群之后可以不再关注频道。 

###  [Matrix](<../zh-cn/Matrix.html> "Matrix") 房间

主群位于 [#archlinux-cn:nichi.co](<https://matrix.to/#/#archlinux-cn:nichi.co>)。另有位于 [#archlinux-cn-space:nichi.co](<https://matrix.to/#/#archlinux-cn-space:nichi.co>) 的空间，包含多个不同话题的房间。 

###  [IRC](<../zh-cn/Arch_%E7%9A%84_IRC_%E9%A2%91%E9%81%93.html> "IRC") 频道

[#archlinux-cn @ libera](<ircs://irc.libera.chat:6697/#archlinux-cn>)

频道位于 [Libera.Chat](<https://libera.chat/>) 的服务器上，地址是：`irc.libera.chat` 端口 6697。请务必使用 TLS/SSL 连接。依用户 IP 的不同，有可能需要先注册并登录账户。由于 IRC 仅支持文本，不支持图片消息和回复引用，因此不推荐使用。IRC 用户的消息都需要经 Matrix 转发才能到达 Telegram 上的用户。 

###  [XMPP](<https://en.wikipedia.org/wiki/XMPP> "wikipedia:XMPP") 群组

**注意：** 由于疏于维护，可能时不时出现故障，不建议使用。并可能在将来完全关停此服务。

这是一个通过机器人实现的群组。使用 XMPP（Jabber）帐号添加`talk@archlinuxcn.org`为好友即可加入。成功加入将收到欢迎信息。由于本群组仅支持文本，不支持图片消息和回复引用，因此不推荐使用。XMPP 用户的消息都需要经 IRC 转发才能到达 Matrix 和 Telegram 上的用户。 

##  论坛

论坛的地址是 <https://forum.archlinuxcn.org/>。注意论坛的账户系统是独立的，与本维基并不互通。 

##  Telegram 频道

[@archlinuxcn](<https://t.me/s/archlinuxcn>) 频道用于发布一些 Arch Linux 新闻的中文翻译、用户需要注意的事项，同时也作为部分其它 Telegram 群组进入的反垃圾手段。提示：即使没有 Telegram 账号，使用浏览器也可以浏览频道内容的。
