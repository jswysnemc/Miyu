Arch Linux中的软件包通过 [makepkg](<../zh-cn/Makepkg.html> "Makepkg") 生成，每一个软件包都有一个构建脚本（称为[PKGBUILD](<../zh-cn/PKGBUILD.html> "PKGBUILD")）。打包后，软件可以通过 [pacman](<../zh-cn/Pacman.html> "Pacman") 安装和管理。[官方仓库](<../zh-cn/%E5%AE%98%E6%96%B9%E4%BB%93%E5%BA%93.html> "官方仓库")中的 PKGBUILD 可以从 [ABS](<../zh-cn/Arch_%E6%9E%84%E5%BB%BA%E7%B3%BB%E7%BB%9F.html> "ABS") 树获得；更多脚本位于（不受官方支持）[Arch 用户软件仓库](<../zh-cn/Arch_%E7%94%A8%E6%88%B7%E8%BD%AF%E4%BB%B6%E4%BB%93%E5%BA%93.html> "Arch 用户软件仓库")。 

在这你可以找到关于如何使用 [pacman](<../zh-cn/Pacman.html> "Pacman") 和 [Arch 编译系统](<../zh-cn/Arch_Build_System.html> "Arch Build System")查找、安装、删除和进行其他管理软件包的操作，. 

**亮点：** [Making a Custom Package CD](</wzh/index.php?title=Making_a_Custom_Package_CD&action=edit&redlink=1> "Making a Custom Package CD（页面不存在）")
