模拟器被用于复制另一个平台或系统的功能，以允许应用程序和游戏在不支持的环境中运行。 

**注意：** 要获得更多最新的模拟器选择，请尝试检索 [AUR 'emulator' 关键字](<https://aur.archlinux.org/packages/?SeB=k&K=emulator>)。

**警告：** 拥有一个高级模拟器是合法的，但是分发任何受版权保护的 ROMs 都是**非法的** 。因此，Arch Linux 并不发布这些受版权保护的内容，包括游戏 ROM 和破解的主机 BIOS。您对使用从[官方仓库](<../zh-cn/%E5%AE%98%E6%96%B9%E4%BB%93%E5%BA%93.html> "官方仓库")或 [Arch User Repository](<../zh-cn/Arch_User_Repository.html> "Arch User Repository") 获得的模拟器产生的任何法律后果负有全部责任。
