[PowerShell](<https://microsoft.com/PowerShell>) 是微软开发的跨平台任务自动化框架和脚本语言。它专为系统管理员和高级用户设计，通过命令行 shell 和脚本语言促进管理任务的自动化，从而实现对 Windows 和其他操作系统的高效管理。

[安装](<../zh-cn/Help:%E9%98%85%E8%AF%BB.html#%E5%AE%89%E8%A3%85%E8%BD%AF%E4%BB%B6%E5%8C%85> "安装") [powershell](<https://aur.archlinux.org/packages/powershell/>)AUR 或者 [powershell-bin](<https://aur.archlinux.org/packages/powershell-bin/>)AUR。 

PowerShell Core 也可以通过 [.NET SDK](<../zh-cn/.NET.html#%E5%AE%89%E8%A3%85_PowerShell_Core> ".NET") 安装（如果已经配置）。
