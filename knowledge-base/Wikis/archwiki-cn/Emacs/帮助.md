Emacs 有一个内置教程，点击启动屏幕上的第一个链接，然后从菜单中选择 _Help >Emacs Tutorial_，或者按 `C-h t`，即可进入该教程。 

要以英语以外的语言（如**简体中文** ）阅读教程，请使用命令 `Alt x`，并输入 `help-with-tutorial-spec-language`。 

Emacs 在设计上是自文档化的。因此，我们可以利用大量信息来确定特定命令的名称或其按键绑定。使用 `C-h C-h` 查看所有上下文帮助绑定。 

Emacs 还包含一套参考资料卡，对初学者和专家都很有用，请参见 `/usr/share/emacs/_version_ /etc/refcards/` 。 

##  手册

如果你真的想掌握 Emacs，最推荐的文档来源仍然是[官方手册](<https://www.gnu.org/software/emacs/documentation.html>)： 

  * Emacs：完整的 Emacs 用户手册。
  * Emacs FAQ。
  * Emacs Lisp Intro：如果你以前从未使用过任何编程语言，那么就来学习 Emacs Lisp 入门吧。
  * Elisp：如果您已经熟悉一种编程语言。

您可以从 [GNU.org](<https://www.gnu.org/software/emacs/manual/>) 或直接从 Emacs 本身通过内嵌的 _info_ 阅读器以 PDF 格式访问它们： `C-h i`。按 `m` 键选择图书。 

一些用户喜欢使用 _info_ 阅读书籍，因为它有方便的快捷键，段落可根据窗口宽度调整，字体可根据当前屏幕分辨率调整。有些人则认为 _info_ 对眼睛的刺激较小。最后，你可以轻松地将书中的内容复制到任何 Emacs 缓冲区，甚至可以直接执行示例中的 Lisp 代码片段。 

您可能想阅读 **Info** 书籍以了解更多：`C-h i m info <RET>`。 在 _Info mode_ 下按 `?` 可快速查看快捷键列表。
