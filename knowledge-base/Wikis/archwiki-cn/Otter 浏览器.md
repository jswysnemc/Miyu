[Otter Browser](<https://otter-browser.org/>) 是旨在重新创建 [Opera](<../zh-cn/Opera.html> "Opera") UI 的 Web 浏览器。它基于 [Qt5](<../zh-cn/Qt.html> "Qt") 和 QtWebKit，并具有 QtWebEngine 的实验支持。 

##  安装

Otter Browser 可以使用以下软件包之一[安装](<../zh-cn/Help:%E9%98%85%E8%AF%BB.html#%E5%AE%89%E8%A3%85%E8%BD%AF%E4%BB%B6%E5%8C%85> "安装"): 

  * [otter-browser](<https://archlinux.org/packages/?name=otter-browser>)包 \- 具有 QtWebKit 和 QtWebEngine 支持的稳定版本
  * [otter-browser-webengine](<https://aur.archlinux.org/packages/otter-browser-webengine/>)AUR \- 没有 QtWebEngine 支持的稳定版本
  * [otter-browser-git](<https://aur.archlinux.org/packages/otter-browser-git/>)AUR \- 最新开发快照

##  提示和技巧

###  对 QtWebEngine 的实验支持

要使用基于 Blink web engine 的 QtWebEngine，请转到 `about:config` 并选择 _qtwebengine_ 作为 _Backend_.
