###  网络剪切板

参见 [Wikipedia:Pastebin](<https://en.wikipedia.org/wiki/Pastebin> "wikipedia:Pastebin")。 

网络剪切板通常用于在协作和故障排除时引用文本或图像。Pastebin 客户端提供了从命令行发布消息的便捷方式。 

**注意：** 通常来说，建议使用不需要JavaScript即可查看、不显示广告、不会操控上传内容、不需要登录的网络剪切板。 [pastebin.com](<https://pastebin.com/>) 已被部分人屏蔽，除去一直以来的使用问题（必须启用JavaScript、显示广告、自行插入CRLF、随机显示验证码）。**非常不建议** 使用该服务。

####  不用专用客户端

一些服务可以与更通用的命令行工具一起使用，例如 [cURL](<../../zh-cn/CURL.html> "CURL")。为了扩展，例如行号，可以使用更多的命令行工具。例如 `cat -n`。 

  * [0x0.st](<https://0x0.st/>) 是一个文件托管服务。用法示例如下：

    $ _command_ | curl -F 'file=@-' 0x0.st

或上传文件: 
    
    $ curl -F 'file=@-' 0x0.st < _file_

**注意：** Read the instructions on the website for further options such as file removal tokens, expiration, and private URLs.

  * [termbin.com](<https://termbin.com>) works with [nc](</wzh/index.php?title=Nc&action=edit&redlink=1> "Nc（页面不存在）").
  * [paste.c-net.org](<https://paste.c-net.org/>) accepts HTTP requests and works with [nc](</wzh/index.php?title=Nc&action=edit&redlink=1> "Nc（页面不存在）").
