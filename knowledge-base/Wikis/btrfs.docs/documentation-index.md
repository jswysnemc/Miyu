# Welcome to BTRFS documentation!

BTRFS is a modern copy on write (COW) filesystem for Linux aimed at implementing advanced features while also focusing on fault tolerance, repair and easy administration. You can read more about the features in the `introduction<Introduction>` or choose from the pages below. Documentation for command line tools `btrfs`, `mkfs.btrfs` and others is in the `manual pages<man-index>`.

## Need help?

Assistance is available from the [\#btrfs channel on Libera Chat](https://web.libera.chat/#btrfs) or the [linux-btrfs mailing list](https://subspace.kernel.org/vger.kernel.org.html). Issues with the userspace btrfs tools can be reported to the [btrfs-progs issue tracker on GitHub](https://github.com/kdave/btrfs-progs/issues).
