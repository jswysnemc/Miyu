Welcome to the EasyEffects wiki!

Most wiki pages contain only EasyEffects information. EasyEffects (>=6.0.0) only supports the PipeWire audio server and is actively maintained.

PulseEffects (<=4.8.7) supports PulseAudio, and has limited older wiki sections in [its own page](https://github.com/wwmm/easyeffects/wiki/PulseEffects-Wiki). PulseEffects is in maintenance mode.

For getting Easy Effect running, you need to install PipeWire. See [here](https://lowtechlinux.com/2022/04/26/go-full-pipewire-on-ubuntu-22-04-and-all-family-derivatives/) for Ubuntu 22.04.
