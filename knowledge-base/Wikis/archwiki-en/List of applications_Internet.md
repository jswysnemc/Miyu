## Pastebin services
See also: Wikipedia:Pastebin

Pastebin services are often used to quote text or images while collaborating and troubleshooting. Pastebin clients provide a convenient way to post from the command line.

## Without a dedicated client
Some services can be used with more general command line tool such as cURL. For extensions, such as line numbers, more command line tools can be used such as .

* 0x0.st is a file hosting service. Usage examples are:

* privatebin.info is a minimalist, open-source online pastebin service where the server has zero knowledge of stored data. It focuses on decentralized hosting and has a directory of public instances.
* termbin.com works with nc.
