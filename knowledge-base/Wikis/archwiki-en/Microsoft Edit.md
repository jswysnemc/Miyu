# Microsoft Edit

Microsoft Edit (or msedit) is a simple text editor written in Rust and aims to small binary.

This editor pays homage to the classic MS-DOS Editor, but with a modern interface and input controls similar to VS Code. The goal is to provide an accessible editor that even users largely unfamiliar with terminals can easily use.

msedit supports features including LF/CRLF conversions, i18n TUI, ability to make use of the mouse and UTF-8 encoding.

The package provides msedit command.
