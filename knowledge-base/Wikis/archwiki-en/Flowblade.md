# Flowblade

Flowblade is a multitrack non-linear video editor released under GPL3 license.

Flowblade supports all media that can be accessed in a Linux system when the FFMPEG library is working as the backend. The typical startup message reports 146 formats, 78 video codecs and 58 audio available codecs.

## Installation
Install the  package.

Additionally, more installation options, such as Flatpak and a development version can be found from Flowblade's download page
