# SDR++

SDR++ is a cross-platform and open source software-defined radio application with the aim of being bloat free and simple to use. It is comparable to the Windows program SDR# (SDRSharp).

## Installation
SDR++ is available in the AUR from . This will build the default modules in this list along with BladeRF, LimeSDR, SDRplay, PortAudio and M17 decoding support.
