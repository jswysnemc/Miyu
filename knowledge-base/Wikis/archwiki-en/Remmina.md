# Remmina

Remmina is a remote desktop client written in GTK hosted on Gitlab It supports the following protocols: SSH, VNC, RDP, X2Go, HTTP(S)

## Usage
Remmina connection profiles are saved connection to connect to your servers, the file name is autogenerated, but you can create manually your own files with the FILE name you prefer.

Profiles can be created through the GUI or manually by placing a  file in .

To connect with a saved profile:
 $ remmina -c /path/to/example.remmina

## Updating profiles from the command line
Profile options can be updated non-interactively. Example:
 $ echo "mypassword" | remmina --update-profile ~/.config/remmina/example.remmina \
   --set-option username=user \
   --set-option resolution_mode=2 \
   --set-option password
