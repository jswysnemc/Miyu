# Fortune

fortune is a simple program that displays random poignant, inspirational, silly or snide phrases from a database of quotations. The fortune command-line utility is part of the  package.

## Installation
Install . Alternatively install , a fortune-mod clone written in haskell.

The fortune-mod package in official Arch Linux repositories doesn't include offensive fortunes but you can get offensive content enabled, if you install the alternative  build from the AUR.
