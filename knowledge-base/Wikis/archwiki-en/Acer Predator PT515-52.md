## Installation
"Optane with RAID" mode is enabled by default. AHCI mode must be used, otherwise the system will not boot.

Changing the storage mode from "Optane with RAID" to AHCI may be challenging as the BIOS does not list the option to do so by default. Going to the main tab in the UEFI menu, and pressing  brings up the relevant option.

## Accessibility
The appearance of the BIOS is simple and colorful, so it may not work well with OCR software. It does not requires the user to use a mouse.
