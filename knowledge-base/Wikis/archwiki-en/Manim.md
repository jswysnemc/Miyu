# Manim

Manim is a library for animating technical concepts using the simplicity of python to generate animations programmatically.

It was originally created by Grant Sanderson (3Blue1Brown) and later forked by the community to create Manim Community Edition. Grant Sanderson's code is now called  and the community's fork is now called  or .
