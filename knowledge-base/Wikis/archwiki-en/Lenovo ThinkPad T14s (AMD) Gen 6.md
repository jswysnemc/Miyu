## Installation
Wi-Fi does work during installation, although connectivity issues have been observed on 5 GHz Wi-Fi channels. Consider using 2.4 GHz in that case.

## Fan / Thermal management
The default thermal management leads to the fan almost always running (seems to be a common issue as also being mentioned in a review).

The good news is that the devices are supported by thinkfan, so this can be mitigated. See Fan speed control.

## WWAN
This Thinkpad model seems to use the same WWAN modules as the Lenovo ThinkPad X1 Carbon (Gen 12). Following instructions on that wiki page works for the 5G module with identifier .

## Accessibility
The BIOS has a newer graphical user interface with pointer navigation. It is still readable and can be navigated with the keyboard. A user manual with explanations is available, see #See also.
