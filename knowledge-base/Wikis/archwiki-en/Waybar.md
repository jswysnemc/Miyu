# Waybar

Waybar is a highly customizable Wayland bar for Sway and Wlroots based compositors. Its configuration is done in JSONC with styling in CSS.

## Installation
Install the  package.

For the icons of the preconfigured modules a font like  is needed.
Some standard modules need further programs to function properly. Those are visible through error messages from those modules.
