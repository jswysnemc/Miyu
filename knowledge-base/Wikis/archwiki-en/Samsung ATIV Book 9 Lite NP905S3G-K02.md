## Installation
Set the firmware to EFI OS and deactivate Secure Boot. If GRUB UEFI will not be loaded after installation and configuration, then the GRUB EFI executable might not at the right location. In this case, the GRUB EFI executable can be moved to the right location on the EFI system partition as described in GRUB#Default/fallback boot path.
