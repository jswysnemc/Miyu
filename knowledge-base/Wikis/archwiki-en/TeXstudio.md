# TeXstudio

TeXstudio is a free LaTeX editor. It offers multiple features to easily input special characters, edit tables, preview math mode snippets and language checking.

## Installation
If not already installed, install TeX Live or similar. This is required for the compiling features to work.

Install the  package.

## Language checking
If you want to enable the language checking functions, install  and at least one of the corresponding dictionaries. The selection of languages available in TeXstudio will match the hunspell dictionaries installed. If no dictionaries are installed, it is impossible to select a language in the application.

You can also install  for a more advanced language check, or connect to a remote LanguageTool server. In any case, you need to install the hunspell dictionary for your desired language, otherwise TeXstudio will not let you select it, even if your LanguageTool installation or server supports it.

## Troubleshooting
## Viewer does not work
Make sure that the output document type matches the viewer type under Options > Configure TeXStudio > Build.
