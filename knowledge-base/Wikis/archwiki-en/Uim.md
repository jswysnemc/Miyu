# Uim

uim (universal input method) is a multilingual input method framework.

## Installation
Install the  package.

## Usage
Add the following to xprofile or xinitrc:

 export GTK_IM_MODULE=uim
 export QT_IM_MODULE=uim
 uim-xim &
 export XMODIFIERS=@im=uim

uim provides several toolbars (), if you want to autostart one you can also do it here.

## Configuration
uim provides several GUI preference editors ().

Choose your preferred input method as Default input method.

You can run  or restart X to test your settings.
