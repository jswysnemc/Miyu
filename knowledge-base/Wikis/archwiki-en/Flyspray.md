# Flyspray

Flyspray is a bug tracking system written in PHP. FlySpray was notably used by Arch Linux itself.

## Installation
Install the  package. Flyspray requires a web server such as Apache HTTP Server with PHP, and an SQL server such as MySQL or PostgreSQL.
