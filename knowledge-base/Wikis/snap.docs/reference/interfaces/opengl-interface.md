#  opengl interface

`opengl` allows access to the OpenGL stack and, consequently, OpenGL hardware, which may be needed for certain types of graphical applications and those that use [CUDA](https://en.wikipedia.org/wiki/CUDA) libraries for computation. See [Adding OpenGL/GPU support to a snap](https://forum.snapcraft.io/t/adding-opengl-gpu-support-to-a-snap/6273) for more information on how to use this interface.
