shorin所有的便利工具都可以通过`shorin-contrib-git`安装上，安装完成后运行shorin命令可以查看帮助信息，运行shorin link命令可以把脚本链接到`~/.local/bin`中。

```
 shorin
用法: shorin <子命令> [选项]

可用子命令:
  change-grub-theme 更改grub主题
  checkallupdates 检查所有待更新软件
  check-battery   显示电池信息
  clean           清理系统
  compressvideos  压缩视频
  getown          获取文件/目录所有权
  lsi             在终端预览图片
  media-info      显示多媒体信息
  mirror-update   更新镜像源
  pac             Pacman/AUR安装软件的TUI
  pacd            降级软件
  pacr            卸载软件包TUI
  pacrrr          追踪软件的残留文件并卸载
  pak             Flatpak安装软件TUI
  preview         预览各种东西
  quickload       快速读档，快照恢复
  quicksave       快速存档，创建快照
  searchmodels    快速搜索AI供应商的可用模型
  sysup           更新系统根
  video2gif       视频转gif
  vir             简单管理KVM虚拟机

环境管理:
  link            生成本地用户的快捷软链接 (全程免密)
  unlink          移除本地软链接
  ```
